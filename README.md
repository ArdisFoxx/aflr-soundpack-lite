# AFLR Soundpack Lite

The recommended companion soundpack for **[ArdisFoxx's Lewd Roleplaying](https://github.com/ArdisFoxx)** (AFLR / AFLP, module id `ardisfoxxs-lewd-pf2e`) on Foundry VTT.

A curated, lightweight selection of voices and SFX. It is a small download that still covers AFLP's full audio feature set, so most users want this one. The full **AFLR Soundpack** (every voice actor + the complete SFX library) is available separately for anyone who wants the lot.

## Install

In Foundry: **Add-on Modules -> Install Module**, paste this manifest URL, and click Install:

```
https://github.com/ArdisFoxx/aflr-soundpack-lite/releases/latest/download/module.json
```

Then enable **AFLR Soundpack Lite** under Manage Modules. AFLP detects it automatically - there is no path to configure. Turn on Voice and/or Ambient SFX in the AFLP settings and assign a voice profile per actor from their AFLP sheet.

Manual install: download `module.zip` from the [latest release](https://github.com/ArdisFoxx/aflr-soundpack-lite/releases/latest), unzip into `Data/modules/`, and restart Foundry.

## Requires

- **ArdisFoxx's Lewd Roleplaying** (`ardisfoxxs-lewd-pf2e`). The soundpack has no effect on its own.

## What's inside

- **Voice profiles** covering the full event set (moans by arousal tier, climax, edge, advance, struggle, defeat, cumflation, oral), with female, male, and creature voices.
- **Ambient SFX** for the categories the engine uses: positional plaps, oral (gluk), fingering (schlick), and the cumflation stings.

If a voice profile is missing a specific event category, AFLP falls back gracefully, so every profile still works.

## Want more?

Install the full **AFLR Soundpack** instead of Lite and AFLP uses it automatically (it takes priority when both are present). It bundles every voice actor and the complete categorised SFX library.
